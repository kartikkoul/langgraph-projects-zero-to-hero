# LangGraph Practice Projects: Zero to Hero
---

## What this repository is about

This repository is a hands-on LangGraph learning journey for beginners.
It is being built over time with multiple projects that progress from beginner to advanced level, so you can build practical depth by studying and implementing real workflows step by step.

Each project introduces one clear real-world use case and builds your understanding step by step:

- Core graph design fundamentals
- Production-style workflow thinking
- Advanced LangGraph patterns through practice

---

## Project #1 - AI Customer Support Workflow

A simple workflow where a user sends a support query and the graph processes it intelligently.

**Example query:**  
`"My payment failed but money got deducted."`

**The graph decides:**
- Is this billing?
- Is this technical?
- Is this urgent?
- Should it retry?
- Should it ask the user again?

**You will practice:**
- Basic `StateGraph` architecture
- Sequential workflows
- Parallel workflows
- Conditional workflows
- Iterative workflows
