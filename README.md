# LangGraph Practice Projects: Zero to Hero

This repository is a hands-on LangGraph learning journey for learners.
It is being built over time with multiple projects that progress from beginner to advanced level, so you can build practical depth by studying and implementing real workflows step by step.

Each project focuses on a clear use case and introduces concepts incrementally, helping you strengthen both fundamentals and advanced LangGraph design patterns.

## Project #1 - AI Customer Support Workflow

A simple LangGraph workflow where a user sends a support query, and your graph processes it intelligently.

Example: “My payment failed but money got deducted.”

Graph decides:
- Is this billing?
- Is this technical?
- Is this urgent?
- Should it retry?
- Should it ask the user again?

Practice: Basic StateGraph Architecture, Sequential Workflows, Parallel Workflows, Conditional Workflows, Iterative Workflows
